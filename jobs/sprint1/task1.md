# Init layout for the mentor/mentee registration page

## User Story
As a developer, I want to implement the structural React layout and skeleton components for the Mentor/Mentee registration flow, so that two separate feature teams can develop the Mentor and Mentee forms in parallel without merge conflicts or architectural blocking.

## Acceptance Criteria
Layout Wrapper: A high-level layout component exists that includes common elements (e.g., Progress Bar, Sidebar, Footer, or Header).
Slot/Component Injection: The layout uses children or specific "slots" so feature teams can plug in their forms without touching the layout code.
Responsiveness: The skeleton layout is responsive to mobile and desktop views.
